Клиенты

Справочник "Клиенты" содержит информацию обо всех частных покупателях с которыми работает компания. Справочник необходим если компания занимается розничной торговлей.В toCase мы делим клиентов на два вида: национальные и зарубежные.

Карточка клиента

Карточка содержит информацию о клиенте. При ее оформлении важно максимально заполнить информацией все поля. В карточке есть редактируемая позиция "Реклама", позволяющая указать источник привлечения.

Как удалить клиента

Если клиент новый, или ранее не фигурировал в документах - его карточку можно удалить, в противном случае удалить клиента нельзя. Если с каким-либо клиентом вы больше не работаете, и не хотите чтобы этот клиент появлялся в списках выбора нужно снять отметку "Активный".

Выгрузка в XLS.

Программа позволяет выгрузить весь список клиентов в файл xlsx. Эта функция сейчас работает только в версии для Windows при установленном Microsoft Excel (тестирование проводилось на версии 2007г). Для успешной выгрузки в папке templates должен находиться файл шаблона temp_client.xlsx.
